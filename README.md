### Running the local project

```bash
# Clone this repository
$ git clone git@github.com:hivecloud-dev/unit-test-academy.git

------------------------------------------------------------------------

# Access the project folder in the cmd/terminal
$ cd unit-test-academy-api

# Install the dependencies
$ npm install

# Run the application in DEVELOPMENT mode
$ npm run start

------------------------------------------------------------------------

# Access the project folder in the cmd/terminal
$ cd unit-test-academy-app

# Install the dependencies
$ npm install

# Run the application in DEVELOPMENT mode
$ ng serve

```
